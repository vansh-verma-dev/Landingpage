# Digital Marketing Service Landing Page

This project is a **responsive Digital Marketing Service Landing Page**
built using **HTML & CSS only** as part of the **CodSoft Internship Task**.

##   Features
- Modern & clean UI design
- Fully responsive (Mobile, Tablet & Desktop)
- Hero section with background image, animation & hover effects
- Mobile-friendly **hamburger navbar** (CSS only)
- Hover effects on buttons and cards
- Smooth CSS animations
- Service-based landing page layout for client conversion

##   Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)

##   Responsive Design
The website adapts perfectly to:
- Mobile devices (with hamburger menu)
- Tablets
- Desktop screens

##   Purpose
This landing page is designed to promote **Digital Marketing Services**
and convert visitors into potential clients through a clean and
professional interface.
